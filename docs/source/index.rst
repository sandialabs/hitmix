HITMIX
******
Computation of hitting time moments using a linear algebraic approach,
as defined in the following paper:

-  *A Deterministic Hitting-Time Moment Approach to Seed-set Expansion over a Graph*, A. Foss, R.B. Lehoucq, Z.W. Stuart, J.D.Tucker, J.W. Berry, `arxiv:2011.09544`_, 2020.
